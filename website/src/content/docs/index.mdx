  <Card title="Typed bus" icon="seti:typescript">

## Get started in three steps

1. Install: `pnpm add @serviceconnect/core @serviceconnect/rabbitmq`
2. Create a bus, register a message type, attach a handler.
3. `await bus.start()` and publish.

The [Getting Started](/ServiceConnect-NodeJS/learn/getting-started/) walkthrough takes you from zero to a running consumer in about ten minutes.
